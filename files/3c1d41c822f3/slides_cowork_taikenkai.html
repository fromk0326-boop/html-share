概要 0:00〜0:15

チャット型AIとCowork、何が違うか

CHAT
チャット型（ChatGPT等）
  • 質問に「答えを返す」
  • 結果は画面の文章。コピペは自分
  • ファイルは1つずつ手で渡す
  • 例えるなら「物知りな相談相手」
COWORK
Claude Cowork
  • 仕事を「終わらせて納品する」
  • 結果はファイル。Excel・Word・PDFで返る
  • フォルダごと渡せば自分で読みに行く
  • 例えるなら「PCの前に座る新人スタッフ」
合言葉：「教えてもらう」から「やってもらう」へ。
概要 0:00〜0:15

Coworkの動き方は4ステップ

ステップ誰が何をする
1. 依頼あなた日本語で仕事を頼む（指示は雑でOK、後で直せる）
2. 計画Cowork「こう進めます」と段取りを提示
3. 実行Coworkフォルダのファイルを読み、作業する
4. 納品Cowork成果物ファイルを作って報告
安心ポイント
  • 触れるのは「許可したフォルダの中」だけ
  • ファイルの削除は必ず確認してくる
  • 途中で「ちょっと待って、こうして」と口を挟める
LIVE
まず見てください
散らかったフォルダを、Coworkが3分で整理します
準備 0:15〜0:25

セットアップ最終確認

チェックリスト（隣の人と確認）
  • デスクトップアプリでログインできている
  • 画面上部に「Chat / Cowork」の切り替えがある
  • Coworkタブを開いてエラーが出ない
  • デスクトップに「cowork練習」フォルダがある（中にファイル20個以上・PDF/Excel/Word/画像が混在）
フォルダを忘れた方へ：配布した練習用サンプル（Driveリンク/QR）をダウンロードすればOKです。未契約の方はお隣の画面で一緒に体験してください。
約束ごと：顧問先の実データ（申告書・マイナンバー・口座情報）は今日のワークでは使わない。
ワーク① 0:25〜0:40

ワーク①：フォルダ整理を任せる（15分）

最初の一歩。「フォルダごと渡す」感覚をつかみます。

デスクトップの「cowork練習」フォルダを見てください。 中のファイルを種類ごとにサブフォルダを作って整理して、 最後に「何をどこに移したか」の一覧表をExcelで作ってください。 ファイルの削除はしないでください。
見てほしいポイント
  • 作業前に「計画」を出してくる → 内容を読んでから承認する
  • フォルダへのアクセス許可を求めてくる → 許可する
  • 終わったら整理結果とExcelができている
ワーク② 0:40〜1:00

ワーク②：請求書PDFをExcel一覧に（20分）

税理士業務に一番近いワーク。「読み取り → 表に整形」はCoworkの得意技です。

「cowork練習」フォルダにあるPDFを全部読んで、 次の列でExcel一覧を作ってください。 列：ファイル名／書類の種類／日付／相手先／金額（税込） 読み取れなかった項目は「要確認」と入れてください。
応用（早く終わった人向け）
  • 「金額の合計行を追加して」
  • 「日付順に並べ替えて、月ごとに小計を入れて」
  • 「源泉徴収・軽減税率8%・外貨建ての請求書だけ抜き出して」
  • 「全セルのフォントをメイリオUIにして」
ここが記帳代行・証憑整理の自動化の入口。事務所で使うイメージが湧くはず。
ワーク③ 1:00〜1:15

ワーク③：顧問先向け案内文書を作る（15分）

「調べて、まとめて、Wordで納品」まで一気通貫でやらせます。

顧問先に送る「年末調整の必要資料のご案内」をWordで作ってください。 条件： ・対象は従業員10名以下の小規模法人 ・必要書類をチェックリスト形式で ・提出期限は11月末と記載 ・A4で1枚、やわらかい敬語で
仕上げの一言（修正指示の練習）
  • 「もう少し簡潔に。チェックリストを表組みにして」
  • 「同じ内容でメール本文版も作って」
一発で完璧を求めない。「6割の下書き → 口頭で直す」が一番速い使い方。
まとめ 1:15〜1:30

任せ方のコツ 3か条

コツ悪い例良い例
1. 成果物を指定する「請求書を整理して」「請求書を読んでExcel一覧にして」
2. 形式・条件を足す「案内文を作って」「A4で1枚、チェックリスト形式で」
3. 直しは会話で最初から完璧な指示文を悩む出てきた物に「ここ直して」と言う
新人スタッフへの頼み方と同じ。最初の指示は6割で出して、対話で詰める。
まとめ 1:15〜1:30
