STEP 3 調整して最終決定

提案をもとに、役割ごとのツール（Cowork／Claude Code）・契約形態・プランを選び直して最終決定します。ツールを切り替えると想定トークン量と推奨プランが連動します。定額プランは「従量課金（API）ならいくら相当か」で選ぶのがコツです。

Cowork と Claude Code の関係: どちらも同じClaude有料プラン（Pro／Max／Team）の中で動き、ツールごとの追加課金はありません。違いは「ワークロードの重さ＝消費トークン量の傾向」と「CLIの要否」。所長は自動化基盤を重く回すのでClaude Code、仕訳・OCR・連絡が中心のスタッフは承認ベースのCoworkで足りる、という棲み分けが目安です。
